# QSP-Protocol
Qwex Secure Protocol (QSP) — лёгкий и современный протокол защищённого канала связи, построенный на основе Noise IK, обмена ключами X25519, шифрования ChaCha20-Poly1305 и хэширования Blake2b
//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
Qwex Secure Protocol (QSP) is a lightweight and modern secure communication protocol built on Noise IK, using X25519 key exchange, ChaCha20-Poly1305 encryption, and Blake2b hashing.
